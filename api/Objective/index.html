Objective

An objective for use in ZS stories. These can be created both on individual clients and on the server.

Properties​

Type​

</>
Objective.Type: string

Type of "kill", "find", "collect", "interact", or "money".

Text​

</>
Objective.Text: string

Text for the objective which will display in UI.

IsPrimary​

</>
Objective.IsPrimary: boolean

Shows the objective as having priority in UI.

MarkerMap​

</>
Objective.MarkerMap: Folder?

A folder of nodes which will be used to guide the player towards Objective.Location.

MarkerPositions​

</>
Objective.MarkerPositions: Vector3 | {Vector3}?

Points to place markers for the objective.

MarkerParts​

</>
Objective.MarkerParts: BasePart | {BasePart}?

Parts to attach objective markers to.

Location​

</>
Objective.Location: Vector3?

The objective location for single goal type objectives. Makes use of MarkerMap to guide the player towards the goal using pathfinding generated markers.

Progress​

</>
Objective.Progress: number?

The current progress for the objective displayed in UI.

ProgressTotal​

</>
Objective.ProgressTotal: number?

The target progress for the objective displayed in UI.

TargetPlayers​

</>
Objective.TargetPlayers: {Player}?

An array of players which will be given the objective.

ImageID​

</>
Objective.ImageID: string?

An image ID which will replace the objective type's icon in UI.

AccentColor​

</>
Objective.AccentColor: Color3?

Overrides the objective type's accent color in UI.

ProgressFormat​

</>
Objective.ProgressFormat: string?

Overrides the objective type's progress format. For a reference on how to use format strings, see format strings. The format string will be used like so: string:format(progress, progressTotal).

AddMarker​

</>
Objective:AddMarker(location: Vector3 | BasePart) → string

Add a new marker on the HUD with the objective's identifier.

AddMarkers​

</>
Objective:AddMarkers(locations: Vector3 | {Vector3} | {BasePart}) → ()

Add markers from vector3s or BaseParts.

AppendPlayer​

This item only works when running on the server. Server
</>
Objective:AppendPlayer(appendingPlayer: Player) → ()

Includes an additional player in Objective.TargetPlayers.

Destroy​

</>
Objective:Destroy(wasCompleted: boolean?) → ()

Destroy the objective.

Remove​

</>
Objective:Remove() → ()

Remove the objective (same as objective:Destroy).

Complete​

</>
Objective:Complete() → ()

Remove the objective and show completion UI.

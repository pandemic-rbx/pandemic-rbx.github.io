ProximityPromptZS

A positional prompt which users may interact with.

Range​

Distance within which the prompt will be visible.

Obstructable​

</>
ProximityPromptZS.Obstructable: boolean

Whether or not line-of-sight is required to see the prompt. Instances in ProximityPromptZS.ObstructionIgnoreList will not block line-of-sight.

</>
ProximityPromptZS.ObstructionIgnoreList: {Instance}

Text displayed on the UI which represents the action being performed.

ObjectText​

</>
ProximityPromptZS.ObjectText: string

Text displayed on the UI which represents the object being acted on.

Enabled​

</>
ProximityPromptZS.Enabled: boolean

Whether or not the prompt is enabled for interaction.
